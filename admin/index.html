Especificações Técnicas das Imagens

Fotos do Portfolio

Resolução Mínima: 1920x1080px (Full HD)
Resolução Ideal: 2560x1440px ou superior
Proporção: 16:9 ou 4:3 (recomendado)
Formato: JPG, PNG, WebP
Tamanho Máximo: 10MB por imagem
Qualidade: 85-95% (para JPG)

Logos dos Apoiadores

Resolução Ideal: 400x200px
Formato: PNG (com fundo transparente)
Tamanho Máximo: 2MB
Proporção: 2:1 (largura:altura)

Dicas Importantes

Compressão: Use ferramentas como TinyPNG para otimizar
Backup: Mantenha sempre cópias originais das imagens
Nomes: Use nomes descritivos (ex: futebol_final_2024.jpg)
Performance: Imagens otimizadas carregam mais rápido

Backup e Restauração

Faça backup dos seus dados ou restaure de um backup anterior.

Informações:

  • Exportar/Importar: Salve ou carregue um backup completo (arquivo JSON).
  • Resetar Galeria: Restaura a galeria para as 9 fotos padrão. As fotos atuais serão perdidas.
  • Limpar Tudo: Apaga TODOS os dados do painel (fotos, mensagens, etc). Ação irreversível.
  • A importação substituirá todos os dados atuais.
